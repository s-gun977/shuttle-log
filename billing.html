🔄 모바일 운행일지 앱에서 가져오기

기사분들이 모바일 운행일지 앱에서 내보낸 JSON 파일을 불러옵니다.
가져온 데이터는 해당 월의 운행일지와 주유내역에 자동 반영됩니다.

📋 가져오기 미리보기

📝 사용 방법
1. 기사님이 모바일 운행일지 앱에서 매일 운행 기록을 입력합니다.
2. 월말에 운행일지 앱 → 내보내기 → "청구서 앱용 파일 내보내기" 클릭
3. 다운로드된 JSON 파일을 이 페이지에서 불러옵니다.
4. 미리보기 확인 후 "가져오기 실행" → 운행일지 & 주유내역 자동 반영
5. 대시보드와 청구서에서 자동 집계됩니다.
