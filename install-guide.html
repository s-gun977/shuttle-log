🚌 운행일지 앱 설치 방법

검암역 로열파크씨티 푸르지오 셔틀버스 기사용

방법 1: 가장 쉬운 방법 (추천)

Android (삼성, LG 등)

1
Chrome 브라우저로 운행일지 앱 주소에 접속합니다
2
화면 하단에 "홈 화면에 추가" 또는 "앱 설치" 배너가 나타납니다
3
"설치" 버튼을 누르면 홈 화면에 "운행일지" 앱 아이콘이 생깁니다
4
홈 화면의 아이콘을 눌러 앱처럼 사용합니다
배너가 안 나타나면: Chrome 메뉴(⋮) → "홈 화면에 추가" 또는 "앱 설치"

iPhone (아이폰)

1
Safari 브라우저로 운행일지 앱 주소에 접속합니다 (반드시 Safari)
2
하단의 공유 버튼 (▶ 네모에서 화살표 나오는 아이콘)을 누릅니다
3
"홈 화면에 추가"를 선택합니다
4
이름을 "운행일지"로 확인 후 "추가"를 누릅니다

방법 2: APK 파일로 설치 (Android 전용)

앱을 웹서버에 올린 후 PWABuilder.com에서 APK로 변환할 수 있습니다.

1
운행일지 파일들을 웹서버에 업로드합니다 (GitHub Pages 추천)
2
PWABuilder.com 접속 → 앱 주소 입력
3
"Android" 선택 → "Generate" 클릭
4
APK 파일 다운로드 → 스마트폰에 전송 → 설치
APK 설치 시 "알 수 없는 앱 설치 허용"을 해야 합니다.
설정 → 보안 → 알 수 없는 소스 허용

방법 3: GitHub Pages로 무료 배포

1
GitHub.com에 가입 (무료)
2
새 저장소(Repository) 생성: 예) shuttle-log
3
운행일지앱 폴더의 파일 5개를 모두 업로드:
index.html, manifest.json, sw.js, icon-192.png, icon-512.png
4
Settings → Pages → Source를 "main" 브랜치로 선택 → Save
5
몇 분 후 https://사용자이름.github.io/shuttle-log/ 주소로 접속 가능
이 주소를 기사분들에게 공유하면 각자 스마트폰에서 "홈 화면에 추가"로 앱 설치 가능!
HTTPS로 제공되므로 PWA 설치 기능이 완벽히 작동합니다.

필요한 파일 목록

  • 📄 index.html - 앱 본체 (25KB)
  • 📄 manifest.json - 앱 설정 정보
  • 📄 sw.js - 오프라인 지원 (Service Worker)
  • 🎨 icon-192.png - 앱 아이콘 (작은)
  • 🎨 icon-512.png - 앱 아이콘 (큰)
중요: 데이터는 각 기사님의 스마트폰에만 저장됩니다.
앱을 삭제하거나 브라우저 데이터를 지우면 기록이 사라집니다.
정기적으로 "관리 → 내보내기"로 CSV 백업을 권장합니다.
